# RAB-Based-App-01
RAG bases application using custom data + Azure AI Search + Azure AI Foundry

RAG based Python application using Azure OpenAI, Azure AI Search and Azure AI Foundry. It starts from provisioning the resources in Azure portal, deploys Azure OpenAI model, create an index using Azure AI search and then utilize everything in a Python application.

1. Custom Data - Taking own data, feeding into the system and then querying this data. Not using any inbuilt datasets or data sources
2. Azure AI Foundry, Azure AI Search for vectorization and LLM is Azure Open AI

Azure AI Foundry + Azure AI Search + Azure OpenAI
